# Event Companion – Project Plan

## Product Vision

Event Companion helps attendees find the right information at the right time, making live events easier and less stressful through AI-powered guidance.

---

# Problem Statement

Attending live events can be overwhelming.

Fans often have questions such as:

- When should I arrive?
- What should I expect at security?
- Is there merchandise?
- Where should I go first?
- What's the best strategy to avoid unnecessary stress?

Information is often scattered across venue websites, emails, social media, and community posts.

Event Companion brings everything together into one simple AI-powered experience.

---

# Target Audience

People attending:

- Concerts
- Sporting Events
- Theater Shows
- Conventions
- Museums
- Other live events

---

# Version 1.0 Scope

## Core Features

- AI-generated Event Guide
- Artist / Event input
- Venue input
- Optional Event Date
- Ticket Type selection
- Follow-up questions
- Guest access (no login required)
- Clean, mobile-friendly interface

---

# UX Principles

- User-first
- Guest-first
- Ask only for essential information
- Keep responses short and practical
- Never invent venue-specific information
- AI supports the experience, not the other way around

# Lessons Learned

## July 2026

- Simplicity beats feature overload.
- Login should be optional.
- AI should enhance the experience, not become the experience.
- Building the right product is as important as building the product right.
