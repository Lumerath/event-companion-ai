# 🎵 Event Companion AI

Event Companion AI is a web application that helps people prepare for concerts and live events using Artificial Intelligence.

Users enter the name of an event, and the application generates a practical event guide with useful tips such as:

- 🎟 Arrival
- 🚪 Entrance
- 👕 Merchandise
- 🍔 Food
- 🚶 Leaving

## 🛠 Technologies

- Python
- Flask
- OpenAI API
- HTML
- CSS
- Git
- GitHub

## ✨ Features

- Generate AI-powered event plans
- Personalized event recommendations
- Error handling for AI requests
- Clean and responsive user interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Lumerath/event-companion-ai.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```
